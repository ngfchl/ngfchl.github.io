# 获取百度 OCR 接口

## 1. 访问登录[百度智能云](https://cloud.baidu.com/) ，无百度账号的先注册账号

## 2. 进入 `人工智能` => `文字识别` 服务

![img.png](/images/baodu-ocr/img.png)


## 3. 创建应用

1. 进入创建应用列表界面，点击`创建应用`
   ![img_1.png](/images/baodu-ocr/img_1.png)

2. 名称随便填，接口部分：文字识别选择全部，归属选择个人

   ![img_2.png](/images/baodu-ocr/img_2.png)

## 4. 获取密钥

   ![image-1669265845876](https://img.ptools.fun/blog/image-1669265845876.png)

## 5. 接下来就可以使用密钥在你的代码中调用文字识别接口了
