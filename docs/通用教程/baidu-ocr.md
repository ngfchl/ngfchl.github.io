# 获取百度 OCR 接口

## 1. 访问登录[百度智能云](https://cloud.baidu.com/) ，无百度账号的先注册账号

## 2. 进入 `文字识别` 服务



![image-1669265768478](https://img.ptools.fun/blog/image-1669265768478.png)

image-1669265768478



## 3. 领取免费资源

1. 进入领取页面

   ![image-1669265927358](https://img.ptools.fun/blog/image-1669265927358.png)

   image-1669265927358

2. 选择

   ```
   通用场景OCR
   ```

   ，选择全部资源，点击

    

   ```
   0元领取
   ```

   ![image-1669266013974](https://img.ptools.fun/blog/image-1669266013974.png)

   image-1669266013974

## 4. 创建应用

1. 进入创建应用界面

   ![image-1669265899474](https://img.ptools.fun/blog/image-1669265899474.png)

   image-1669265899474

2. 名称随便填，接口部分：文字识别选择全部，归属选择个人

   ![image-1669265869980](https://img.ptools.fun/blog/image-1669265869980.jpeg)

   image-1669265869980

## 5. 获取密钥



![image-1669265845876](https://img.ptools.fun/blog/image-1669265845876.png)

image-1669265845876
