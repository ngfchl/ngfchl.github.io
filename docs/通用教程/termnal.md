# 各种NAS进入容器命令行

## 群晖进入终端

1. 在容器详情中进入终端机，点击新增，即可在右侧输入命令，之后从上方第三步开始

## 威联通进入终端

1. 威联通进入容器界面后，点击终端机即可进入终端界面，之后也从上方第三步开始
